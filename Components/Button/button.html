Buttons

A Button is used to trigger an event or action. It allows user to make choices,with a single tap. They are usually used with other components such as forms, dialogs, cards, toolbars,etc.

Contained Buttons

A Contained button comes with two colors primary and secodary. They are the basic buttons which takes primary actions of your app

tooltip 
          
            <button class="button btn-primary">primary</button>
            <button class="button btn-secondary">secondary</button>

Link Buttons

Links allow users to click their way from page to page. There are two variants of links as below.

LINK LINK
tooltip 
          
            <a class="button link">LINK</a>
            <a class="link-secondary">LINK</a>

Icon Buttons

Icon buttons are commonly found in app bars and toolbars. Sometimes you might want to have icons for certain buttons to enhance the UX of the application. So, we have provided two variants depending on your need.

tooltip 
          
            <a class="button icon"><i class="fas fa-trash-alt fa-lg"></i></a>
            <button class="button btn-icon"><i class="fas fa-bolt"></i> Buy now</button>

Floating Action Buttons

A floating action button (FAB) performs most common or primary action on the screen. It is usually circular in shape with icon in its center.

Note: Only one component is recommended per screen to represent the most common action.
tooltip 
          
            <button class="button fab">
              <i class="fas fa-plus"></i>
            </button>
            <button class="button fab-secondary">
              <i class="fas fa-pen"></i>
            </button>
