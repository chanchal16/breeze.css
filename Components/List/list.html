Stacked list

Stacked list have items stacked. It is mainly used to create notifications. To use this component just add .stacked-list to the <ul> element.

  • Notification 1
  • Notification 2
  • Notification 3
  • Notification 4
tooltip 
                  
                    <ul class="list stacked-list">
                      <li class="stacked-list-item">Notification 1</li>
                      <li class="stacked-list-item">Notification 2</li>
                      <li class="stacked-list-item">Notification 3</li>
                      <li class="stacked-list-item">Notification 4</li>
                    </ul>
