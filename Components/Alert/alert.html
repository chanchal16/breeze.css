Alerts

An alert is used to display short, important message to attract user's attention with minimal user interaction.

Simple alert

Alert comes with four severity levels with a distinctive icon and color. Alerts can be created using alert class followed by the severity name.

This is basic alert!

This is success message!

This is error message.

This is warning message.

tooltip 
        
          <div class="alert basic">
            <p>This is basic alert!</p>
          </div>
          
          <div class='alert success'>
            <span><i class="far fa-check-circle"></i></span>
            <p>This is success message!</p>
          </div>
          
          <div class='alert error'>
            <i class="fas fa-skull"></i>
            <p>This is error message.</p>
          </div>
          
          <div class='alert warning'>
            <i class="fas fa-exclamation-triangle"></i>
            <p>This is warning message.</p>
          </div>

Alerts with action

Alert can have a closing action button. It is rendered after the message at the end of the alert.
