Avatars

The Avatar component is used to represent a user or an entity (such as project, repository, or space), and displays the profile picture, initials or fallback icon.

Rounded Avatars

There are three sizes available --small, medium, and large you can access this using classes .sm, .md, .lg respectively. 

                  
                    <img class="avatar sm" src="https://avatars.dicebear.com/api/bottts/x4.svg" alt="avatar1" />
                    <img class="avatar md" src="https://i.pravatar.cc/150?img=37" alt="avatar2" />
                    <img class="avatar lg" src="https://avatars.dicebear.com/api/avataaars/x18.svg" alt="avatar3" />

Avatars with Initials

You can also create avatars using initials of users. You can use these avatars as default if user does not provide a picture or if there is an error loading the src of the avatar.

CR
CR
CR
                  
                    <div class="avatar sm">CR</div>
                    <div class="avatar md">CR</div>
                    <div class="avatar lg">CR</div>

Square Avatars

If you need a square variant you can simply add .square class to the existing class. 

                  
                    <img class="avatar square sm" src="https://avatars.dicebear.com/api/bottts/x4.svg" alt="avatar" />
                    <img class="avatar square md" src="https://i.pravatar.cc/150?img=37" alt="avatar" />
                    <img class="avatar square lg" src="https://avatars.dicebear.com/api/avataaars/x18.svg" alt="avatar" />
