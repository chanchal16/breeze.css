Slider

A range slider is an input field that can be used to select a numeric value within a given range. They are probably used for videos,audios, price in shopping site, etc.

Simple list

tooltip 
                    
                        <input type="range" min="0" max="100" value="50" />

Styled slider

There are two variants. You can use them by simply adding class .accented or .slider

.
tooltip 
                    
                        <input class="accented" type="range" min="0" max="100" value="50" />
                        <input class="slider" type="range" min="0" max="100" value="50" />
