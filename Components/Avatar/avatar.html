Avatars

The Avatar component is used to represent a user or an entity (such as project, repository, or space), and displays the profile picture, initials or fallback icon.

Rounded Avatars

There are three sizes available --small, medium, and large you can access this using classes .sm, .md, .lg respectively.

Avatars with Initials

You can also create avatars using initials of users. You can use these avatars as default if user does not provide a picture or if there is an error loading the src of the avatar.
