Badge

Badges are used to highlight an item's status for quick recognition. They small circles usually positioned at top-right or bottom right of the parent element.

Badge on icon

There are two types of badges available for icon --number badge and dot badge you can access this using classes .numbadge and .dotbadge respectively followed by .badge class.

Profile badge is used to show the current status of the user. There are three variants for the status namely-- online, offline, busy.
